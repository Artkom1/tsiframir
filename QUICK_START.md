# ⚡ QUICK START - ВКЛЮЧИТЬ AI НА VERCEL (10 минут)

## 🎯 ВСЕ ЧТО НУЖНО СДЕЛАТЬ

### **ШАГ 1: Получить OpenAI ключ (вы уже это сделали!)**

Ваш ключ готов. Держите его под рукой.

---

### **ШАГ 2: Получить Vercel Auth Token**

```
1. Откройте: https://vercel.com/account/tokens
2. Нажмите "Create Token"
3. Заполните:
   - Token name: numerology-setup
   - Expiration: 7 days
4. Нажмите "Create"
5. СКОПИРУЙТЕ ТОКЕН (выглядит: vercel_xxx...)
```

---

### **ШАГ 3: Запустить одну команду в терминале**

```bash
bash /home/mikhail/numerology-forum-landing/vercel-deploy.sh
```

**Скрипт спросит вас:**
1. Vercel Auth Token (введите из Шага 2)
2. OpenAI API Key (введите ваш ключ)
3. Имя проекта (просто нажмите Enter)

**После ввода скрипт сделает ВСЁ автоматически! ✅**

---

## ✨ ЧТО ПРОИЗОЙДЕТ

```
1. Скрипт установит Vercel CLI
2. Скрипт создаст проект на Vercel
3. Скрипт свяжет GitHub репозиторий
4. Скрипт добавит переменные окружения
5. Скрипт развернет backend на Vercel
6. Скрипт выдаст вам URL сайта
7. Deployment займет 1-3 минуты

```

## 🧪 ТЕСТИРОВАНИЕ

После того как deployment закончится:

1. **Откройте URL** которую выдаст скрипт
2. **Введите дату рождения** (например: 11, 09, 2012)
3. **Нажмите "Рассчитать"** → локальный анализ (0.1 сек) ✅
4. **Нажмите "✨ Получить профессиональный AI анализ"**
5. **Подождите 5-10 секунд** (идет запрос к ChatGPT)
6. **Появится красивый анализ!** 🤖
