## 🔐 ПОТОКИ ДАННЫХ И БЕЗОПАСНОСТЬ

### **Где хранится API ключ?**
```
❌ НЕ в браузере (видно пользователю)
❌ НЕ в HTML (видно в исходнике)
❌ НЕ на GitHub (публично)

✅ В Vercel Settings → Environment Variables
   (Защищено, недоступно клиенту)

✅ На сервере (backend) в переменной окружения
   (Используется только при отправке в Claude API)
```

### **Информационные потоки**
```
Браузер пользователя:
├─ Видит: Число, дата, имя (ввод пользователя)
├─ Видит: Результаты расчетов
├─ Видит: Локальный анализ
├─ Видит: AI анализ (если есть)
└─ НЕ видит: API ключ ✓

Backend сервер:
├─ Получает: Число, тип (от браузера)
├─ Использует: API ключ (из env)
├─ Отправляет: Промпт + число (в Claude API)
├─ Получает: Анализ (от Claude)
├─ Сохраняет: В кэш (в памяти сервера)
└─ Возвращает: Анализ браузеру ✓

Claude API (Anthropic):
├─ Получает: Промпт + контекст (от backend)
├─ Генерирует: Профессиональный анализ
└─ Возвращает: Текст анализа ✓

GitHub:
├─ Видит: Весь код
├─ НЕ видит: API ключ (в .env local) ✓
└─ Видит: vercel.json (конфиг без ключа) ✓
```
