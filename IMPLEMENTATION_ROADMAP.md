```json

{

      "relationships": "...",
      "health": "..."
    },
    // ... для всех 59 чисел
  }
}
```

### **2. Backend (Этап 2)**
```javascript
// api/ai-interpretation.js — основной endpoint

export default async function handler(req, res) {
  // 1. Проверить API ключ
  if (!process.env.ANTHROPIC_API_KEY) {
    return res.json({ success: false, fallback: true });
  }

  // 2. Проверить кэш
  const cached = getFromCache(req.body.number);
  if (cached) return res.json({ success: true, analysis: cached });

  // 3. Отправить в Claude
  const analysis = await callClaudeAPI(req.body);

  // 4. Сохранить в кэш
  saveToCache(req.body.number, analysis);

  // 5. Вернуть результат
  return res.json({ success: true, analysis });
}
```

### **3. Frontend (Этап 3)**
```javascript
// calculator-ui.js — добавить обработчик кнопки

button.addEventListener('click', async () => {
  const response = await fetch('/api/ai-interpretation', {
    method: 'POST',
    body: JSON.stringify({ number, type })
  });

  const result = await response.json();

  if (result.success) {
    displayAIAnalysis(result.analysis);
  } else {
    showLocalAnalysis(); // fallback
  }
});
```

## 🚀 ШАГ ЗА ШАГОМ

### **ШАГ 1: Получить API ключ (15 минут)**
```bash
# 1. Зайти на https://console.anthropic.com
# 2. Нажать "Create API Key"
# 3. Скопировать ключ вида: sk-ant-v4-xxxxxxxxxxxxx
# 4. Сохранить где-нибудь в безопасном месте
```

### **ШАГ 2: Подготовить локальные данные (2-3 часа)**
```bash
# 1. Открыть assets/data/numerology-system.json
# 2. Расширить каждое число (1-59) с подробными описаниями
# 3. Использовать вашу книгу "Оцифрованный мир" как источник
# 4. Добавить психологию, карьеру, отношения, рекомендации
# 5. Проверить что JSON валидный
```

### **ШАГ 3: Создать backend (2-3 часа)**
```bash
# 1. Создать папку: mkdir api && mkdir api/utils
# 2. Создать 4 файла:
#    - api/ai-interpretation.js
#    - api/utils/claude-client.js
#    - api/utils/prompts.js
#    - api/utils/cache.js
# 3. Скопировать код из HYBRID_AI_PLAN.md
# 4. npm install @anthropic-ai/sdk
```

### **ШАГ 4: Обновить frontend (1.5 часа)**
```bash
# 1. Обновить calculator-ui.js:
#    - Добавить renderLocalInterpretation()
#    - Добавить requestAIAnalysis()
#    - Добавить displayAIAnalysis()
#    - Добавить обработчик для fallback
# 2. Добавить CSS стили для AI анализа
# 3. Протестировать в браузере
```

### **ШАГ 5: Развернуть (1 час)**
```bash
# 1. Создать .env.local:
#    ANTHROPIC_API_KEY=sk-ant-xxxxx

# 2. Создать vercel.json

# 3. git add -A && git commit && git push

# 4. Зайти на vercel.com
# 5. Подключить GitHub репозиторий
# 6. В Settings → Environment Variables добавить API ключ
# 7. Deploy автоматически

# 8. Тест: Зайти на https://tsiframir.ru
#    Нажать [AI Анализ] → Должен работать
```
