# 🚀 VERCEL SETUP - ПОЛНАЯ АВТОМАТИЗАЦИЯ

## 📋 ЧТО НУЖНО СДЕЛАТЬ (Всего 3 шага!)

### **ШАГ 1: Получить OpenAI ключ (2 минуты)** ✅ У вас уже есть!

Ваш ключ:
```
sk-proj-... (тот что вы создали)
```

Скопируйте его и держите готовым!

---

### **ШАГ 2: Получить Vercel Auth Token (1 минута)**

1. **Откройте ссылку:**
   ```
   https://vercel.com/account/tokens
   ```

2. **Нажмите "Create Token"**

3. **Заполните форму:**
   ```
   Token name: numerology-setup
   Expiration: 7 days (или больше)
   ```

4. **Нажмите "Create"**

5. **Скопируйте токен** (выглядит как: `vercel_xxx...`)

---

### **ШАГ 3: Запустить наш автоматический скрипт**

```bash
# Скопируйте и запустите эту команду в терминале:

bash /home/mikhail/numerology-forum-landing/vercel-deploy.sh
```

Скрипт попросит вас ввести:
1. **Vercel Auth Token** (из Шага 2)
2. **OpenAI API Key** (у вас уже есть)
3. **Vercel Project Name** (можно оставить `numerology-forum-landing`)

После ввода скрипт сделает ВСЁ АВТОМАТИЧЕСКИ:
- ✅ Создаст проект на Vercel
- ✅ Свяжет с GitHub
- ✅ Добавит переменные окружения
- ✅ Развернет backend
- ✅ Настроит всё

---

## 🎯 ВАЖНЫЕ МОМЕНТЫ

**Vercel Auth Token:**
- Не делитесь им!

## ✅ ПОСЛЕ СКРИПТА

После того как скрипт закончит:
