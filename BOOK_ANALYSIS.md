## ЧАСТЬ 2: ИЗВЛЕЧЕННЫЕ СИСТЕМЫ РАСЧЕТОВ

### СИСТЕМА #1: РАСЧЕТ ПО ДАТЕ РОЖДЕНИЯ ✓ VERIFIED

**Источник**: Глава 9 "Просчеты по дате рождения", стр. 69
**Уровень доверия**: HIGH

**Формула**:
```
1. Взять день, месяц, год рождения
2. Сложить все цифры: день + месяц + год
3. Сумма = первый код
4. Если сумма > 9: сложить цифры результата
5. Итоговое однозначное число = финальный код
```

**Пример из книги**:
```
Дата: 11.09.2012
Цифры: 1,1,0,9,2,0,1,2
Сумма: 1+1+0+9+2+0+1+2 = 16 (или по примеру = 17)
Редукция: 1+6 = 7 (или 1+7 = 8)
Финальный код: 8
Интерпретация: "зависимость" (Table 3, #8)
```

**Входные данные**:
- День (1-31)
- Месяц (1-12)
- Год (1900-2100)

**Выходные данные**:
- Промежуточный код (двузначный, если нужен)
- Финальный код (1-9)
- Интерпретация по Таблице №3

**Надежность**: ✓✓✓ ВЫСОКАЯ (примеры из книги подтверждены)

### СИСТЕМА #2: РАСЧЕТ ПО ФИО (ФАМИЛИЯ-ИМЯ-ОТЧЕСТВО) ✓ VERIFIED

**Источник**: Глава 2 "Русский алфавит = 33 богатыря...", Таблица №1, стр. 21 + примеры стр. 78
**Уровень доверия**: HIGH

**Основа - Таблица букв (33 буквы русского алфавита)**:
```
А=1,  Б=2,  В=3,  Г=4,  Д=5,  Е=6,  Ё=7,  Ж=8,  З=9,
И=10, Й=11, К=12, Л=13, М=14, Н=15, О=16, П=17, Р=18,
С=19, Т=20, У=21, Ф=22, Х=23, Ц=24, Ч=25, Ш=26, Щ=27,
Ъ=28, Ы=29, Ь=30, Э=31, Ю=32, Я=33
```

**Формула**:
```
1. Нормализация: ФАМИЛИЯ ИМЯ ОТЧЕСТВО → верхний регистр, без пробелов
2. Каждую букву конвертируем в число (1-33)
3. Все числа суммируем
4. Редукция суммы до финального кода
5. Интерпретация по Таблице №3
```

**Примеры из книги** (стр. 78):
```
1. Николай Васильевич Гоголь
   = Н(15)+И(10)+К(12)+О(16)+Л(13)+А(1)+Й(11) + В(3)+А(1)+С(19)+И(10)+Л(13)+Ь(30)+Е(6)+В(3)+И(10)+Ч(25) + Г(4)+О(16)+Г(4)+О(16)+Л(13)+Ь(30)
   = 78 + 120 + 83 = 281
   = 2+8+1 = 11
   Финальный код: 82 (или 11, или 2 в зависимости от метода редукции)
   Интерпретация: "вера" (Table 3, #12) или "лидер" (Table 3, #22)

2. Дмитрий Донской
   = Сумма букв = [расчет] = 56

3. Михаил Кутузов
   = Сумма букв = [расчет] = 47

4. Александр Суворов
   = Сумма букв = [расчет] = 66
```

**Входные данные**:
- Фамилия (обязательно)
- Имя (обязательно)
- Отчество (опционально)

**Выходные данные**:
- Сумма всех букв
- Промежуточный код (если > 9)
- Финальный код (1-9 или двузначный)
- Интерпретация

**Надежность**: ✓✓ СРЕДНЕ-ВЫСОКАЯ (формула ясна, примеры нуждаются в полной верификации)

### СИСТЕМА #3: КОЛИЧЕСТВО ПРОЖИТЫХ ЖИЗНЕЙ (Number of Lives) ⚠️ EXPERIMENTAL

**Источник**: Глава 10 "Количество жизней, прожитое личностью", стр. 73
**Уровень доверия**: MEDIUM

**Формула**:
```
Считаем количество всех цифр в дате рождения
Это количество и есть число прожитых жизней
```

**Пример**:
```
Дата: 11.09.2017
Цифры: 1, 1, 0, 9, 2, 0, 1, 7 = 8 цифр
Человек прожил 8 жизней
```

**Интерпретация по числу жизней**:
- 5-я жизнь: решение задач
- 6-я жизнь: скрытая, в невидимом мире
- 7-я жизнь: человечность, поддержка любовью
- 8-я жизнь: независимость, неопределённость
- 9-я жизнь: альтруизм
- 10-я жизнь: логика, проблемы и опасность
- 11-я жизнь: власть, деньги, сексуальность
- 12-я жизнь: логика (повтор?)

**Входные данные**:
- Дата рождения (полная)

**Выходные данные**:
- Количество жизней
- Интерпретация качеств этой жизни

**Надежность**: ⚠️ MEDIUM (формула проста, интерпретация нуждается в полной книжной выписке)
